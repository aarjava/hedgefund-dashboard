## 2024-05-24 - [Financial Input Clarity]
**Learning:** Users often struggle with financial jargon like "basis points" (bps) or statistical terms like "quantiles" in input fields. Providing concrete examples (e.g., "10 bps = 0.10%") in tooltips significantly reduces cognitive load and prevents input errors.
**Action:** Always add `help` tooltips with calculation examples for any input field requiring domain-specific knowledge or unit conversion.
