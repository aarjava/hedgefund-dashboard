## 2024-10-24 - Concrete Examples for Abstract Metrics
**Learning:** For inputs involving statistical or financial parameters (e.g., quantiles, basis points), standard labels or generic definitions in tooltips aren't enough. Users often struggle to map input values to real-world impact.
**Action:** Include concrete examples in the `help` tooltips (e.g., "10 bps = 0.10%") for all abstract numeric inputs.
