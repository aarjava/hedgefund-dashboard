## 2024-05-23 - [Technical Terminology Transparency]
**Learning:** Users (even within specialized domains like finance) benefit significantly when technical parameters like "basis points" or "quantiles" include immediate, concrete examples (e.g., "10 bps = 0.10%"). This reduces cognitive load and calculation errors.
**Action:** Always add `help` tooltips with unit conversions and concrete examples for any input involving statistical thresholds or financial units (bps, volatility, etc.).
