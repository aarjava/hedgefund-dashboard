## 2024-05-23 - Financial Jargon Tooltips
**Learning:** Financial applications often use domain-specific jargon (e.g., "Basis Points", "ADV", "Quantile") that can alienate non-expert users. Providing immediate context via tooltips reduces cognitive load and prevents users from needing to consult external documentation.
**Action:** When adding configuration inputs for specialized domains, always include a `help` parameter with a plain-English explanation or concrete example (e.g., "10 bps = 0.10%").
