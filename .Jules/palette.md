Critical UX/accessibility learnings only.

## 2026-01-18 - Financial Parameter Context
**Learning:** Users often assume financial acronyms (bps, quantiles) are universally understood. Adding `help` tooltips with concrete examples (e.g., "10 bps = 0.10%") is a critical accessibility pattern for this domain.
**Action:** Always include concrete conversion examples in tooltips for financial units.
