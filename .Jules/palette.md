## 2024-05-24 - HedgeFund Dashboard UX Init
**Learning:** This app is a professional research tool for quants. Users value density, precision, and financial clarity.
**Action:** Focus on making complex data more digestible without dumbing it down. Look for opportunities in tooltips, empty states, and better feedback for long-running calculations.

## 2024-05-24 - Financial Parameter Clarity
**Learning:** Financial parameters (bps, quantiles) require concrete examples in tooltips to prevent user error. Even "experts" appreciate knowing if "10" means 10bps or 1000bps.
**Action:** Always include a "e.g., X = Y" format in tooltips for numerical inputs with specific units.
