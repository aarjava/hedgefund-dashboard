## 2024-03-14 - Tooltips for Financial Parameter Inputs
**Learning:** Inputs involving statistical or financial parameters (e.g., quantiles, basis points) lack clarity for many users when not accompanied by concrete examples.
**Action:** Always include `help` tooltips containing concrete examples (e.g., "10 bps = 0.10%") to any statistical or financial parameter inputs to make the application more accessible and self-explanatory.
